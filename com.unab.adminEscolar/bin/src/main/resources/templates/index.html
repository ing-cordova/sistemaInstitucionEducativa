¡Hey, te damos la bienvenida a nuestra página web oficial!
Bien, te explicaremos un poco como funciona nuestro sistema para la correcta administración de un centro educativo.

Este proyecto ha sido diseñado, con el fin de dar una solución a las muchas instituciones administrativas escolares, pero te preguntarás ¿que es lo que realmente puedo hacer?, este sistema lleva un control para los 2 principales roles dentro de una institución educativa, tales como un Docente y un Estudiante. Resumiendo, un Docente tiene la facultada de ejecutar algunas de las siguientes tareas: Crear una nueva tarea, Revisar las tareas, inscribir las materias a impartir, etc. Un Estudiante tendría las facultades de enviar las tareas que se le han sido asignadas, visualizar las notas, inscribir materias, etc.
